# Pokedex
Frontend SPA con Vue.js

## Stack
- Vue.js
- Nuxt
- SASS
- Vuetify
- Axios (requests)
- Jest (testing)
- Prettier and Linter

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# run tests with jest
$ npm run test

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
